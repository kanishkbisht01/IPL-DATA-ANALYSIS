# üèè IPL Data Analysis Project (2008-2017)

##  Important Note
**This dataset is synthetic/fictional** - It is not real IPL data. The data has been estimated/assumed for learning and practice purposes only.

##  Project Overview
This project performs data analysis on synthetic IPL cricket data from 2008 to 2017. Various tools and technologies have been used to extract insights and create visualizations.

##  Purpose
- Practice data analysis techniques
- Learn Python, SQL, and Power BI integration
- Understand data visualization concepts
- Build a portfolio project

##  Tools Used
- **Python** - Data cleaning and processing
- **SQL** - Querying and data manipulation  
- **Power BI** - Interactive dashboard creation
- **Git/GitHub** - Version control and hosting

##  Project Structure
- `IPL_Analysis.pbix` - Power BI dashboard
- `Figure_1.png` - Sample visualization
- `PYTHON/CSV.PY` - Python data processing script
- `SQL/IPL-ANALYSIS.sql` - SQL queries
- `README.md` - Project documentation

##  Disclaimer
The data used in this project is **not official IPL data**. It has been created for educational purposes to demonstrate data analysis skills.

## üöÄ How to Use
1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Explore the dashboard and visualizations
4. Check Python scripts and SQL queries
